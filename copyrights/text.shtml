Citing this web site in a bibliography

If you wish to cite the The Milton Reading Room web site in a bibliography, please list it with the URL, and the month and year you accessed it, as in the following example:

Luxon, Thomas H., ed. The Milton Reading Room, http://www.dartmouth.edu/~milton, March, 2008.
